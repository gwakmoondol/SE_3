requirement list (fuctional requirements)


5. 자전거 등록/조회/삭제 기능

-관리자는 자전거 정보를 등록할 수 있다. (ID, 제품명, 유형, 소속 대여소, 상태)
-관리자는 등록된 자전거 목록을 조회할 수 있다.
-관리자는 특정 자전거 정보를 상세 조회할 수 있다.
-관리자는 특정 자전거를 삭제할 수 있다.

6. 대여소 검색 기능

-회원은 대여소 이름으로 대여소를 검색할 수 있다.
-검색 결과로 조건에 맞는 대여소 리스트를 출력한다.

7. 상세정보 조회 및 자전거 즉시 대여/예약대기 기능

-회원은 검색된 대여소를 선택해 상세 정보를 조회할 수 있다. (대여소 이름, 위치, 사용 가능 자전거 목록 등)
-사용 가능한 자전거가 있는 경우 즉시 대여할 수 있다.
-사용 가능한 자전거가 없는 경우 예약대기를 신청할 수 있다.
-즉시 대여 또는 예약대기 시 문자 알림을 수신한다.


use case discriptions (step by step breakdown)


5-(1). 자전거 등록 - actor : admin / description : 관리자가 새 자전거를 시스템에 등록한다.
로그인 - '자전거 정보 등록' 선택 - 자전거 ID, 제품명, 유형, 소속 대여소, 상태 정보 입력 - '등록'버튼 클릭 - 시스템의 입력정보 저장 후 등록 완료 알림 출력

5-(2). 자전거 조회 - actor : admin / description : 관리자가 등록된 자전거 목록 조회 후 상세 정보를 확인한다.
로그인 - '자전거 목록 조회' 선택 - 시스템은 모든 자전거 정보 리스트 출력 - 관리자가 특정 자전거 항목 선택 - 시스템은 해당 자전거의 상세 정보 출력

5-(3). 자전거 삭제 - actor : admin / description : 관리자가 특정 자전거를 삭제한다.
로그인 - '자전거 목록 조회' 선택 - 시스템은 모든 자전거 정보 리스트 출력 - 관리자가 삭제할 자전거 항목 선택 - '삭제' 클릭 - 시스템은 해당 자전거 정보 삭제 후 완료 알림 출력

6. 대여소 검색 기능 - actor : guest (물론 해당 guest는 로그인을 해야만 하는 상태이므로 member로 칭해도 무방) / description : 회원이 대여소를 이름으로 검색한다.
로그인 - 대여소 이름으로 대여소 검색 - 시스템은 조건에 맞는 대여소 목록 출력

7. 상세정보 조회 및 자전거 즉시 대여/예약대기 기능 - actor : guest / description : 회원이 대여소 상세 정보를 확인하고 자전거를 대여하거나 예약한다.
로그인 - 대여소 이름으로 대여소 검색 - 해당 대여소 선택 - 시스템은 해당 대여소의 상세 정보(이름, 위치, 사용 가능 자전거 목록 등) 출력 - [자전거가 있는 경우] guest는 '즉시 대여' 클릭 - 시스템은 자전거 대여 완료 후 완료 알림 출력 / [자전거가 없는 경우] guest는 '예약 대기' 클릭 - 시스템은 예약 정보 저장 후 완료 알림 출력
